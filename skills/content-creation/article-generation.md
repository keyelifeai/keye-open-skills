# 文章生成技能

## 技能描述

快速生成高质量的深度分析文章，适用于技术、商业、创业等主题。

---

## 技能内容

```
你是一位经验丰富的内容创作者，擅长撰写深度分析文章。

请根据以下要求生成文章：

## 核心要求
1. **结构清晰** - 使用清晰的标题层级和段落划分
2. **数据支撑** - 引用具体数据和案例（如果有）
3. **洞察独到** - 提供独特视角，避免陈词滥调
4. **实用性强** - 给出可执行的建议或启发
5. **语言风格** - 专业但不晦涩，有温度但不煽情

## 文章结构
- **标题** - 吸引但不夸张，准确概括核心观点
- **导语** - 100-200字，快速切入主题
- **正文** - 3-5个小标题，每个论点有支撑
- **结语** - 总结观点，给出展望或建议

## 写作风格
- 避免使用"首先、其次、最后"等陈词滥调
- 使用具体例子和比喻来解释复杂概念
- 保持段落简短（3-5句话）
- 使用加粗强调关键观点

## 字数
1500-2500字

请根据我的具体主题和要求进行调整和优化。
```

## 使用方法

### 步骤 1: 复制技能内容
复制上面的技能内容到 Claude Code

### 步骤 2: 添加具体信息
```
主题：[你的文章主题]
角度：[切入角度]
目标读者：[读者画像]
特殊要求：[如有]
```

### 步骤 3: 执行生成
Claude 会根据技能要求和你的具体信息生成文章

### 步骤 4: 后续优化
- 根据需要进行修改
- 添加具体数据和案例
- 调整语言风格

## 优化建议

### 如果输出太泛泛
添加约束：
```
请避免：
- 宏观论述
- 陈词滥调
- 缺乏数据支撑的观点

请确保：
- 每个观点有具体案例
- 引用真实数据（如果有）
- 给出可执行建议
```

### 如果输出不够深入
添加要求：
```
深度要求：
- 不只是描述现象，要分析原因
- 不只是列举问题，要给出解决方案
- 不只是泛泛而谈，要具体到可执行步骤
```

## 常见问题

**Q: 技能需要每次都完整输入吗？**
A: 建议创建模板，根据具体主题调整核心部分即可。

**Q: 如何确保输出符合自己的风格？**
A: 添加"风格参考"部分，提供你的文章样本或描述。

**Q: 输出质量不稳定怎么办？**
A: 检查输入是否清晰，考虑分步生成（先大纲，再填充）。

**Q: 能否生成特定领域的文章？**
A: 在提示词中添加领域背景和要求，效果会更好。
