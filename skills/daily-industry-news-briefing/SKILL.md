# Daily Industry News Briefing

Professional daily news briefings for livestock, feed, and animal nutrition.

## Quick Start

1. Search latest info via WebSearch (past 24-48h per section)
2. Synthesize into 7-section structure
3. Generate formatted Markdown

## Sections

1. Policy & Industry Data - MARA announcements, statistics
2. Enterprise Dynamics - Strategies, M&A, launches
3. Market Trends - Costs, demand, efficiency
4. Pet Industry - Nutrition trends, functional foods
5. Macroeconomic Context - Commodity prices, policies
6. Global Perspectives - Trade, supply chains
7. Reference Sources - Fact-checking sites

## Principles

- Steady, professional tone
- Data-driven analysis
- Multi-angle perspective
- Long-term value focus
